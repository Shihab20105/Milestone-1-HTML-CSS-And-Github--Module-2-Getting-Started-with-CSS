href-Hypertext reference:

What is the HTML a href attribute?

In HTML the inline a (anchor) element denotes a hyperlink from one web address to another. All functional a elements must contain the href(Hypertext reference)attribute indicates the destination of the hyperlink.

src-Source:

The src attribute specifies the URL or path of the resource file. The URL points to the media file to display (images) or play (audio and video). The browser uses the URL to locate and load the media file. The URL can be internal (from same website) or external (another website).

URL-Uniform Resource Locator:

A URL is another word for a web address.

A URL is a unique identifier used to locate a resource on the Internet. It is also referred to as a web address.

A URL can be composed of words (e.g. w3schools.com), or an Internet Protocol(IP) address (e.g. 192.68.20.50).

Simple Selectors.

Simple Selectors.

id- Identity Document in CSS:

A CSS id selector uses the id attribute of an HTML element on a page. To use an id selector in CSS, you simply write a hashtag (#) followed by the id of the element.

class- in CSS:

A CSS class is a attribute used to define a group of HTML elements in order to apply unique styling and formatting to those elements with CSS.

* Universal selector in CSS:

It is usually written as an asterisk followed by a selector. The * is used for selecting all elements. This asterisk also has the capability to select all elements that are inside another element.

Difference between id and class in CSS:

While an ID is specific to a single element, classes can be assigned to multiple elements on a page or throughout the website. They are not unique. And while a single element can only have one ID, it can have multiple classes.
